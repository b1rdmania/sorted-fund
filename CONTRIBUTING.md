# Contributing

Thanks for contributing to Sorted.fund.

## Development setup

1. Install dependencies:
   - `cd backend && npm install`
2. Start backend:
   - `npm run dev`
3. Serve dashboard locally:
   - `cd ../frontend/dashboard-v2 && python3 -m http.server 8081`

## Pull requests

- Keep PRs focused and small when possible.
- Include testing notes in the PR description.
- Do not commit secrets, private keys, or production credentials.

## Reporting security issues

Please do not open public issues for vulnerabilities.
Use the process in [`SECURITY.md`](SECURITY.md).
