What Happens

1. User Clicks Button

No wallet popup. No transaction signing. Just a simple click.

2. Sorted.fund Sponsors Gas

Our backend authorizes the transaction and pays for the gas.

3. Transaction Executes

The transaction is submitted and confirmed on-chain.

4. User Pays Nothing

Zero gas fees. Zero wallet complexity.
